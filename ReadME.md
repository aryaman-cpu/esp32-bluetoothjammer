# ESP32 NRF24 BLUETOOTH JAMMER.
A simple bluetooth jammer using esp-32 and nrf24.

## Features

* ESP32 based
* NRF24L01+ PA/LNA support
* Low latency
* Suitable for robotics and IoT projects

## Hardware

* 1 × ESP32 Development Board
* 1 × NRF24L01+ PA/LNA Module
* 1 × External Antenna
* Jumper Wires
* USB Data Cable

## Wiring

| NRF24 | ESP32   |
| ----- | ------- |
| VCC   | 3.3V    |
| GND   | GND     |
| CE    | GPIO 4  |
| CSN   | GPIO 5  |
| SCK   | GPIO 18 |
| MOSI  | GPIO 23 |
| MISO  | GPIO 19 |

## Setup

1. Connect the NRF24 modules according to the wiring table.
2. Connect both ESP32 boards to your computer using USB data cables.
3. Go to [this website](https://esp32-bluejammerflasher.pages.dev/)
4. Click on Combo Channel option.
5. Select your port and then click on install.
6. Your Esp-32 Bluetoth Jammer is READY.
